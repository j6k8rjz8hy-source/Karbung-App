☁️ Link otomatis ke Google Sheets

Sekali disambungkan, setiap karangan yang disimpan otomatis masuk ke Google Sheets — fotonya diunggah ke Drive dan link-nya ikut tercatat di sheet.

Cara menyambungkan (±3 menit, sekali saja)
  1. Buka/buat Google Sheet untuk pencatatan, lalu menu Extensions → Apps Script.
  2. Hapus isi editor, tempel kode di bawah (tombol salin), lalu simpan.
  3. Klik Deploy → New deployment, pilih tipe Web app. Set Execute as: Me dan Who has access: Anyone, lalu Deploy dan izinkan akses saat diminta.
  4. Salin Web app URL, tempel di kolom di atas, klik Uji koneksi, lalu Kirim yang belum tersinkron.

Catatan: kalau app dibuka lewat halaman pratinjau Claude, koneksi keluar diblokir — pakai versi GitHub Pages (atau buka file index.html langsung) agar sinkron otomatis jalan.

Ekspor manual

CSV berisi kolom yang sama dengan sheet kalian: tanggal penerimaan, jenis, pengirim, ditujukan ke, grouping. Buka file CSV lalu copy‑paste ke Google Sheets.

Impor

Impor CSV dari Google Sheets (File → Download → CSV) atau pulihkan file backup dari HP lain. Data digabung, tidak menimpa.

Pengaturan

Data tersimpan di HP/browser ini (offline, tanpa server). Lakukan backup berkala dan simpan filenya ke Google Drive.
